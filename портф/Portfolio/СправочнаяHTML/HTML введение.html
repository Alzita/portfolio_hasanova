Основные понятия.

  • Гиперссылка-«активная» ссылка на другой документ.
  • Гипертекст-текст,содержащий гиперссыылки
  • Гипермедия-документ, включающий текст,рисунки, звуки, видео, в котором любой элемент может быть гиперссылкой.
  • WWW(World Wide Web)-«всемирная паутина»,система для обмена информацией в виде гипертекста (и гипермедиа).
  • Браузер-программа для просмотра Web-страницы на экране(Internet Explorer, Yandex,Opera).
  • HTML(HyperText Markup Language)-язык разметки гипертекста-предназначен для создания Web-страниц.

Типы Web-страниц.

  • Статические-существует на сервере в виде готовых файлов: *.htm, *.html .
  • Динамические-полностью или частично создаются на сервере в момент запроса(выбор информации из базы данных) *.shtml, *.asp, *.pl, *.cgi, *.php.

К статическим типам относится:


  • Flash-страницы-создаются с использованием технологии разработки анимационных изображений Adobe Flash, а потому такие страницы привлекательные, яркие,содержат много анимации и звуковых эффектов.

К динамическим типам относится:


  • Комбинированные страницы-в которых используются вышеизложенные технологии создания сайтов.

Тег


Тег-это команда языка HTML, которую выполняет браузер:

Непарные теги:

  • Скобки
  • Название тега
  • Содержимое




Парные теги: (контейнеры)

  • Открывающий тег
  • Закрывающий тег
  • Содержание
  • Результат
